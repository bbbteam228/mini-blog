Mini-blog

Aplikacja: artykuły + komentarze + odpowiedzi (mini-blog)

Opis projektu

Mini-blog pozwala na dodawanie artykułów, komentowanie artykułów oraz dodawanie odpowiedzi na komentarze.

Funkcjonalności

  • Lista artykułów (tytuł + krótki opis)
  • Dodawanie artykułu (tytuł, treść, autor)
  • Widok szczegółowy artykułu (pełna treść + komentarze)
  • Dodawanie komentarzy (autor + treść)
  • Dodawanie odpowiedzi na komentarze (reply)
  • REST API (endpointy dla artykułów, komentarzy i odpowiedzi)

Instrukcja uruchomienia (lokalnie)

  1. Sklonuj repozytorium: git clone <repo-url>
  2. Przejdź do katalogu projektu: cd mini-blog
  3. Zainstaluj zależności: npm install
  4. Uruchom serwer: npm start
  5. Otwórz w przeglądarce: http://localhost:3000
